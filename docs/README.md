# DayZ Sentinel — Documentation Hub

Welcome to the central documentation of **DayZ Sentinel**, a FastAPI-based REST API for managing, monitoring, and analyzing DayZ Standalone server data.

## 📋 Contributing to Documentation

1. Use Markdown for all documentation files.
2. Keep technical facts up to date with code changes.
3. Reference sprint IDs (e.g. `SPR-020`) in all relevant files.
4. Follow the [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) format for `CHANGELOG.md`.
5. Add an ADR in `decisions/architecture_decisions.md` for any significant architectural choice.
